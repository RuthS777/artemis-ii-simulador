# Registro Inicial de Riesgos — Equipo Solaris

**Fecha de creación:** 17/05/2026

---

## Descripción

Este registro documenta los riesgos identificados por el equipo que podrían afectar el desarrollo del proyecto. Para cada riesgo se indica su probabilidad de ocurrencia, el impacto que tendría y el plan de acción correspondiente.

---

## Escala de valoración

| Nivel | Probabilidad | Impacto |
|---|---|---|
| Alto / Alta | Muy probable que ocurra | Consecuencias graves para el proyecto |
| Medio / Media | Podría ocurrir | Consecuencias manejables |
| Bajo / Baja | Poco probable | Impacto menor o recuperable |
